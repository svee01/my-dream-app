Entiteiten

Er is nog maar 1 entiteit volgens de deelopdracht. Deze entiteit is voor een Movie. In de Angular web app kan je verschillende movies aanmaken en bekijken. Hier zijn Movies voor nodig. Zie hieronder de Movie entiteit met toepassend plaatje.

Movie

  • _id: number
  • title: string
  • year: number
  • rated: string
  • released: date
  • runtime: number
  • countries: array:string
  • genres: array:string
  • director: string
  • writers: array:string
  • actors: array:string
  • plot: string
  • poster: string
