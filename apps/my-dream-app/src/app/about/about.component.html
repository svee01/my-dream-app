Entiteiten

Er zijn 2 entiteiten volgens de deelopdracht. Deze entiteiten zijn User en Movie. In de Angular web app kan je verschillende users aanmaken en bekijken. Hier zijn Users voor nodig. Hetzelfde geldt voor Movies. Zie hieronder de User en Movie entiteit met toepassend plaatje.

User

  • id: string
  • firstName: string
  • lastName: string
  • emailAddress: string
  • gender: Gender
  • birthDate: Date

Movie

  • _id: string
  • title: string
  • year: int
  • rated: string
  • released: Date
  • runtime: int
  • countries: Array
  • genred: Array
  • director: string
  • actors: Array
  • writers: Array
  • plot: string
  • poster: string
